# Button Jiggler

HID-приложение: Flipper представляется USB-клавиатурой и периодически нажимает выбранную кнопку, чтобы ПК не уходил в сон.

По умолчанию — **пробел раз в 4 минуты**.

## Зачем (vs Mouse Jiggler)

- **Mouse Jiggler** двигает курсор — может сбить прицеливание в игре, дизайн-софте или при перетаскивании
- **Button Jiggler** нажимает на клавишу — может что-то напечатать или вызвать действие
- Идеально использовать вместе с **Shift / F15** (опции в меню) — они не вызывают видимых эффектов, но ОС считает их активностью

## Управление

| Кнопка | Действие |
|--------|----------|
| OK | Включить / выключить |
| ← / → | Переключить редактируемое поле (клавиша / интервал) |
| ↑ / ↓ | Менять значение выбранного поля |
| Back | Выход (восстановит USB-режим) |

## Выбор клавиши

| Опция | Особенности |
|-------|-------------|
| **Space** | Напечатает пробел в текстовом поле. **Внимание:** в YouTube / Spotify нажмёт Play/Pause; в Проводнике может вызвать Quick Look (Mac); в PDF — пролистает страницу |
| **Shift** | Модификатор без символа — ничего не печатает, ничего не вызывает. **Рекомендую** |
| **F15** | Редкая функциональная клавиша — почти ни одна программа на неё не реагирует. **Самый безопасный вариант** |

Если оставляешь окно в фокусе текстового редактора и хочешь видеть «активность» — выбирай Space. Если боишься что-то сломать — Shift или F15.

## Интервалы

15с / 1м / **4м** (по умолчанию) / 6м / 10м.

Большинство OS-таймаутов сна — 5-10 минут. Жми чаще чем таймаут — например 4 минуты для 5-минутного screensaver'а.

## Совместимость

| ОС | Работает |
|----|----------|
| Windows 10/11 | ✅ |
| macOS | ✅ |
| Linux | ✅ |

HID-клавиатура — универсальный USB-стандарт, поддерживается всеми ОС из коробки.

## Сборка

```bash
cd button_jiggler
ufbt
```

Категория: `apps/USB/`.
