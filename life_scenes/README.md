# Life Scenes

Движок мультишаговых сценариев. Одна кнопка запускает цепочку действий: IR-команды, паузы, вибро, бипы — последовательно. Превращает Flipper в программируемый пульт повседневных рутин («Кино», «Сон», «Утро»).

## Как работает

1. Кладёшь `.scene`-файлы в **`/ext/life_scenes/`** на SD
2. Открываешь Life Scenes → видишь список сцен
3. Выбираешь → OK → Flipper выполняет шаги по очереди, показывая прогресс

Папка создаётся автоматически при первом запуске.

## Формат `.scene` файла

Текстовый файл, одна команда на строку. Строки с `#` — комментарии.

```
# Movie night scene
NAME,Movie Night
IR,Samsung32,0x07,0x02
DELAY,1500
IR,NEC,0x04,0x08
VIBRO,200
BEEP,1200,150
```

### Доступные команды

| Команда | Синтаксис | Что делает |
|---------|-----------|------------|
| `NAME` | `NAME,<текст>` | Имя сцены (отображается при запуске) |
| `IR` | `IR,<протокол>,<address>,<command>` | Шлёт ИК-команду (3 повтора) |
| `DELAY` | `DELAY,<мс>` | Пауза |
| `VIBRO` | `VIBRO,<мс>` | Вибрация |
| `BEEP` | `BEEP,<частота_Гц>,<мс>` | Короткий бип через динамик |
| `SUBGHZ` | `SUBGHZ,<файл.sub>` | ⚠️ Зарезервировано, в v1 **пропускается** |

### Поддерживаемые IR-протоколы

`NEC`, `NECext`, `NEC42`, `NEC42ext`, `Samsung32`, `RC5`, `RC5X`, `RC6`, `SIRC`, `SIRC15`, `SIRC20`, `Kaseikyo`, `RCA`, `Pioneer`

Адрес и команда — в hex (`0x07`) или десятичных.

## Где взять IR-коды

Самый простой способ — записать свои пульты:

1. На Flipper'е: `Infrared → Learn New Remote` → нажми кнопку на пульте
2. Сохрани, потом открой `.ir` файл (через qFlipper) — там увидишь `protocol`, `address`, `command`
3. Перенеси эти значения в `.scene`

Или возьми из готовых баз IR-кодов (Flipper IRDB на GitHub).

## Примеры

В папке [`examples/`](./examples) три готовых сцены:

- **movie.scene** — включает ТВ, ждёт, переключает вход, бип «готово»
- **sleep.scene** — выключает ТВ и кондиционер, вибро на прощание
- **goodmorning.scene** — включает ТВ-новости, кондиционер, бодрый бип

⚠️ **IR-коды в примерах — для Samsung/generic.** Скорее всего НЕ подойдут к твоей технике как есть — замени `address`/`command` на коды своих пультов.

## Управление

| Кнопка | В списке | Во время выполнения |
|--------|----------|---------------------|
| ↑ / ↓ | Выбор сцены | — |
| OK | Запустить (или пересканировать если пусто) | — |
| Back | Выход | Прервать сцену |

## Установка примеров

Через qFlipper скопируй файлы из `examples/` в `/ext/life_scenes/` на SD.

## Ограничения v1 / планы

- **SubGHz-шаги пока пропускаются** — инлайн-передача `.sub` требует полного transmit-пайплайна субгерца. Это следующий шаг (нужно тестировать на реальном устройстве — твоих воротах/розетке)
- **HID-шаги** (запереть ПК, нажать хоткей) — тоже план; требует переключения USB в режим клавиатуры посреди сцены
- До 48 шагов на сцену, до 24 сцен
