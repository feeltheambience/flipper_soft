# BLE VPN Button

Беспроводная кнопка управления Windows VPN через **Bluetooth LE**. Flipper не подключён по USB — связь по BLE GATT, на ПК слушает Python-демон.

## Архитектура

```
┌──────────┐   BLE GATT Serial    ┌──────────┐    PowerShell   ┌─────────┐
│ Flipper  │ ─────────────────→  │  Daemon  │ ──────────────→ │ Windows │
│  OK key  │  "TOGGLE\n"          │ (bleak)  │  rasdial /      │   VPN   │
│  L/R     │  "DOWN\n"/"UP\n"     │          │  Connect-Vpn... │         │
└──────────┘                      └──────────┘                 └─────────┘
```

При запуске FAP перенастраивает Flipper'овский BLE-стек с дефолтного **RPC-протокола** (через который работают qFlipper и мобильное приложение) на **сырой Serial GATT** под нашу управление.

При выходе — восстанавливает дефолтный профиль (qFlipper и mobile снова работают).

## Установка

### На Flipper

```bash
cd ble_vpn_button
ufbt launch
```

Категория: `apps/Bluetooth/`.

### На ПК (Windows)

```powershell
cd ble_vpn_button\pc_daemon
pip install -r requirements.txt
python button_daemon.py "ТочноеИмяТвоегоVPN"
```

Перед первым запуском **спарь Flipper с ноутом по Bluetooth** обычным образом:
- Параметры → Bluetooth и устройства → Добавить устройство → Bluetooth
- Должен появиться `Flipper XYZ` — выбери и подтверди код

После пары демон сам подключается каждый раз.

## Управление

| Кнопка | Команда → демон | Действие на PC |
|--------|-----------------|----------------|
| **OK** | `TOGGLE\n` | Если VPN подключён — отключить, иначе подключить |
| **←** | `DOWN\n` | Принудительно отключить |
| **→** | `UP\n` | Принудительно подключить |
| **Back** | — | Выход (восстанавливает дефолтный BLE) |

## Состояния на экране

| Статус | Что значит |
|--------|-----------|
| `Idle` | BLE стек не поднят — что-то не так с Bluetooth настройками |
| `Advertising` | Flipper виден другим устройствам, ждёт коннекта |
| `CONNECTED` | Демон на ПК подключился, кнопки работают |

## Зависимости PC-демона

- **Python 3.10+**
- **bleak** — кросс-платформенная BLE-библиотека (Windows / macOS / Linux)
- **Windows 10/11** с включённым Bluetooth (для BLE)
- Прав администратора **не нужно** для BLE, но **может потребоваться** для `Connect-VpnConnection` системного VPN

## Замечания

- Подключение по BLE требует **пары** между Flipper'ом и ноутом (один раз)
- Если активна qFlipper desktop — отключи на время использования (он держит BLE/USB канал)
- При запуске FAP **разрывает текущий BLE-коннект** (если qFlipper-мобайл был подключен) и поднимает свой профиль. На выходе — наоборот
- Если демон отвалился — реконнект сам каждые 5 секунд

## Протокол

Flipper → Демон, ASCII-строки с `\n`:
```
TOGGLE
UP
DOWN
```

Демон в ответ ничего не шлёт (одностороннее общение). При желании можно расширить — Flipper тогда покажет статус VPN на экране.
