# Alarm Clock

Простой будильник с громкой сиреной и вибрацией. Любая кнопка во время звонка — мгновенный стоп.

## Что делает

- Показывает текущее время и установленное время будильника
- В заданный час:минуту запускает «ringing»:
  - Сирена через встроенный динамик — переключение между **1400 Гц** и **900 Гц** каждые ~180 мс на максимальной громкости
  - Включается **вибромотор** на всё время звонка
  - Подсветка дисплея принудительно включена
- При нажатии **любой** кнопки во время звонка — глушит динамик, выключает вибро и **снимает будильник с охраны** (не сработает повторно сам)

## Управление

| Кнопка | В режиме ожидания | Во время звонка |
|--------|-------------------|-----------------|
| ↑ / ↓ | Изменить выбранное поле (часы/минуты) | Стоп |
| ← / → | Переключение часы ↔ минуты | Стоп |
| OK | Включить / выключить будильник | Стоп |
| Back | Выход из приложения | Стоп |

## Ограничения

- **Работает только пока приложение запущено** — FAP не может работать в фоне. Чтобы будильник сработал ночью, оставь Flipper подключённым к зарядке с открытым приложением. Подсветка сама притухнет
- Только **один** будильник, без снуза и расписаний по дням
- Время берётся из системного RTC Flipper'а — если оно сбито, поправь в `Settings → System → Date and Time`

## Сборка

```bash
cd alarm_clock
ufbt
```

Готовый FAP появится в `dist/alarm_clock.fap`. Категория установки: `apps/Tools/`.
