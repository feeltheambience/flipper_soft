# IR Kill TV

Универсальный «выключатель» телевизоров. По одной кнопке шлёт power-команды для 14 популярных брендов подряд — где-то да выстрелит.

Хорошо работает в барах, аэропортах, торговых центрах и шумных коворкингах. Используй по совести.

## Что делает

При нажатии OK запускается воркер-поток, который проходит по списку брендов и для каждого:

1. Формирует `InfraredMessage` с правильным протоколом, адресом и командой `power`
2. Передаёт через ИК-светодиод Flipper'а **3 раза** (для надёжности — некоторые телевизоры ждут повтор)
3. Пауза 80 мс между брендами

Полный цикл ~5 секунд. UI показывает прогресс: текущий бренд и счётчик `N / 14`.

## Управление

| Кнопка | В idle / done | Во время «Firing...» |
|--------|---------------|----------------------|
| OK | Начать пальбу | — |
| Back | Выход | Прервать |

## Замечания

- **Не все коды попадут в каждый телевизор** — производители используют разные команды на разных рынках, для разных моделей одной серии и т.д. Если конкретный телевизор не выключился — добавь его правильный код в массив
- Эффективность сильно зависит от того, насколько прямо ты целишься ИК-окошком Flipper'а в приёмник телевизора. Светодиод узконаправленный, шейся в радиусе ~5 метров
- Это работает **только с устройствами, у которых одна кнопка одновременно `power on` / `power off`**. Если телевизор использует раздельные команды — может или включить выключенный, или ничего не сделать

## Сборка

```bash
cd ir_killtv
ufbt
```

Готовый FAP появится в `dist/ir_killtv.fap`. Категория установки: `apps/Infrared/`.

## Юридическая оговорка

Используй ответственно. Выключать чужие телевизоры в публичных местах — в большинстве юрисдикций не преступление (тебе ничего за это не будет), но это **не повод устраивать хаос в людных местах**. В частных помещениях без разрешения владельца — уже хулиганство.
