# Vibe Idle Keeper

Умный anti-idle для долгих AI-сессий с Claude / Cursor / Copilot. Похож на Mouse Jiggler, но:

- **Рандомизация интервалов** (±20-30%) — не выглядит как робот для систем поведенческого анализа
- **Невидимые действия по умолчанию** (Shift не печатает символ, ничего не ломает в твоём чате)
- **Пресеты под workflow** — AI Generation / Coding / Watching / Aggressive
- **Чередование Shift + Mouse** — два типа активности, ещё естественнее

## Пресеты

| Пресет | Интервал | Действие | Когда использовать |
|--------|----------|----------|-------------------|
| **AI Generation** | 21-39 сек | Shift или Mouse случайно | Claude/Cursor генерирует ответ 5+ минут |
| **Coding** | 45-75 сек | Только Shift (невидимо) | Активно печатаешь, не хочешь сбить мышь |
| **Watching** | 90-150 сек | Только Mouse | YouTube/Netflix, слабая активность |
| **Aggressive** | 12-18 сек | Shift + Mouse | Параноидальные таймауты (Cisco AnyConnect и т.п.) |

## Управление

| Кнопка | Действие |
|--------|----------|
| OK | Старт / стоп |
| ← / → | Переключить пресет |
| Back | Выход |

## Почему Shift вместо Space

`Space` в браузере прокручивает страницу, в YouTube/Spotify — pause, в Cursor — может вставиться в чат. `Shift` — модификатор без видимого эффекта. Можешь спокойно печатать в чате с AI, jiggle не вставит символы.

## Совместимость

Стандартный HID — Windows / macOS / Linux работают «из коробки».

## Сборка

```bash
cd vibe_idle_keeper
ufbt
```
