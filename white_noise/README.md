# White Noise

Генератор фонового шума через встроенный динамик Flipper'а, с таймером отключения для засыпания. Не нужен телефон под подушкой.

## Звуки

| Пресет | Характер | Для чего |
|--------|----------|----------|
| **White** | Яркий ровный шум (300–6000 Гц) | Заглушить разговоры, офис |
| **Pink** | Мягче, теплее (200–3000 Гц) | Универсальный сон |
| **Brown** | Глубокий низкий гул (80–1200 Гц) | Самый «тёплый», маскирует басы соседей |
| **Fan** | Ровный гул вентилятора | Кто привык спать под вентилятор |
| **Ocean** | Волны — медленные накаты громкости | Расслабление, медитация |
| **Rain** | Редкие капли-щелчки | Дождь за окном |

> Технически Flipper'овский динамик умеет только тоны, поэтому «шум» получается быстрым случайным перебором частот в заданной полосе. Звучит как настоящий hiss/гул, хотя это не сэмплированный звук.

## Таймер сна

Off / 15 / 30 / 45 / 60 / 90 минут. Когда таймер истекает — звук плавно прекращается, на экране «Sleep timer done».

По умолчанию **30 минут**.

## Управление

| Кнопка | Действие |
|--------|----------|
| OK | Вкл / выкл звук |
| ← / → | Сменить тип шума |
| ↑ / ↓ | Изменить таймер сна |
| Back | Выход |

Таймер можно менять на лету во время воспроизведения — отсчёт сразу пересчитается.

## Замечания

- **Громкость** Flipper'а ограничена — это не замена нормальной колонке, но для тихой комнаты у изголовья достаточно
- Динамик пищит на одном тоне очень быстро переключаясь — на максимальной громкости Brown/Fan звучат лучше всего
- Приложение **держит экран включённым** (FAP не работает в фоне) — подсветка сама притухнет, но Flipper должен оставаться с открытым приложением. Поставь на зарядку на ночь
