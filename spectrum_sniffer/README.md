# Spectrum Sniffer

Простой RSSI-монитор по 4-м основным ISM-частотам. Помогает быстро понять, на какой частоте сейчас идёт активность.

## Что делает

В фоне крутится воркер-поток, который по кругу:

1. Переключает CC1101 в `idle`
2. Меняет частоту на следующую из списка: **315 / 433.92 / 868.35 / 915 МГц**
3. Включает RX и ждёт ~25 мс пока АРУ устаканится
4. Читает RSSI в dBm
5. Обновляет данные для отрисовки

UI рисует 4 столбика:

- **Сплошная заливка** — текущий RSSI (-100 dBm в самом низу, -30 dBm на верху шкалы)
- **Горизонтальная чёрточка** над столбиком — пиковое значение, медленно опускается (~0.4 dBm за проход)
- Снизу под каждым столбиком — частота в МГц и текущее значение в dBm

## Управление

| Кнопка | Действие |
|--------|----------|
| OK | Сбросить пиковые значения |
| Back | Выход |

## Как читать показания

- **−100 ... −90 dBm** — тишина, фоновый шум
- **−80 ... −60 dBm** — что-то слабое в окружении
- **−60 ... −40 dBm** — кто-то рядом активно передаёт
- **−40 ... −20 dBm** — пульт или брелок прямо в руке

Хороший тест: возьми любой пульт от ворот / автосигнализации на 433 МГц, нажми кнопку рядом с Flipper'ом — столбик на 434 МГц должен резко подскочить.

## Технические детали

- Использует `subghz_devices` API (а не legacy `furi_hal_subghz_*`) — это новый канонический интерфейс в Unleashed
- Пресет модуляции: `FuriHalSubGhzPresetOok650Async` (OOK, BW 650 кГц) — широкая полоса нужна чтобы ловить активность любого протокола, а не только OOK
- Полный обход 4-х частот занимает ~100 мс, UI перерисовывается каждые 80 мс

## Ограничения

- **Только 4 фиксированные частоты.** Если интересна другая (например, 169 МГц для LoRa-метеостанций) — поправь массив `freqs_hz[]` в `spectrum_sniffer.c`
- **CC1101 не охватывает всё подряд** — поддерживает диапазоны 300–348, 387–464, 779–928 МГц. За них вылезать бессмысленно
- В России регуляторно разрешены только 433.92 (LPD) и 868 (SRD). Это приложение **только слушает** (RX), ничего не передаёт — так что ничего не нарушает

## Сборка

```bash
cd spectrum_sniffer
ufbt
```

Готовый FAP появится в `dist/spectrum_sniffer.fap`. Категория установки: `apps/Sub-GHz/`.
