# RDP Manager

BadUSB-приложение: меню из 4 действий для управления Remote Desktop на Windows. По OK переключает Flipper в режим USB-клавиатуры, открывает elevated PowerShell через `Win+X → A → Alt+Y` и печатает нужный набор команд.

## Меню

| Действие | Что делает |
|----------|-----------|
| **Enable RDP** | Включает службу TermService, открывает правило фаервола, ставит на автозапуск. Выводит USER / HOST / IP для подключения |
| **Disable RDP** | Обратное действие: останавливает службу, закрывает фаервол, ставит в Manual |
| **Add RDP User** | Создаёт локального пользователя `flipperuser` с паролем `RDP_Flipper_2024!` и добавляет в группу `Remote Desktop Users`. Если пользователь уже есть — сбрасывает пароль |
| **Show Status** | Показывает состояние службы, активные правила фаервола, текущего юзера, IP-адреса |

## Логика работы

1. На Flipper'е выбираешь действие → подтверждаешь
2. Flipper переключает USB в режим HID-клавиатуры (`furi_hal_usb_set_config(&usb_hid, ...)`)
3. Ждёт до 5 сек пока PC увидит новое устройство
4. Жмёт `Win+X` → `A` → `Alt+Y` — открывается **PowerShell от админа**
5. Печатает однострочный PowerShell-скрипт с нужными командами
6. Жмёт `Enter` → команды выполняются
7. Flipper восстанавливает прежний режим USB

## Учётные данные

**Скрипт сам не создаёт пароли для существующих пользователей.** RDP использует обычные Windows-учётки.

- **Enable RDP** — после включения заходи под своим **обычным Windows-логином/паролем** того ПК
- **Add RDP User** — создаёт нового юзера `flipperuser` / `RDP_Flipper_2024!`. Логин можно менять в коде (`SCRIPT_ADDUSER`)

Если у твоей учётки нет пароля (только PIN/Windows Hello) — RDP не пустит. Сначала задай обычный пароль через `net user <имя> *`.

## Как подключиться после Enable RDP

С другой машины:
```
mstsc.exe              # Windows
# или Microsoft Remote Desktop / Remmina
```
- **Computer:** IP или hostname из вывода скрипта
- **Username:** твой обычный Windows username (или `flipperuser` если выбрал Add User)
- **Password:** соответствующий пароль

## Требования

- На целевом ПК — **права администратора** у текущей учётки
- Windows 10 или 11 (Win+X → A → ALT+Y работает на обеих)
- Раскладка клавиатуры — EN/US (если активна RU, символы вроде `\` `'` `;` напечатаются неверно — переключи на EN перед запуском)

## Безопасность

RDP с торчащим в интернет 3389 портом = главная цель ботов. Включать **только** в:
- Домашней локалке (за роутером)
- Через VPN до офиса/дома
- Через **Tailscale** / **ZeroTier** / **Cloudflare Tunnel** (рекомендую)

Никогда не пробрасывай порт 3389 напрямую в интернет.

## Сборка

```bash
cd rdp_manager
ufbt
```

Категория установки: `apps/USB/`.

## Юридическая оговорка

Запускать **только на ПК где у тебя явное разрешение** (своих, рабочих с согласия админа, в своей пентест-лабе). На чужом ПК без разрешения — в РФ это **ст. 272 УК**, в США — **CFAA**, до 10 лет. Серьёзно.
